# software-Defined-Networks
Traffic monitoring and flow analysis in Software Defined Networks (SDN) using Mininet, POX Controller, OpenFlow, and Python.

## 📌 Overview

This project implements a **Traffic Monitoring and Flow Analysis System** using **Software Defined Networking (SDN)**. It leverages the **POX Controller**, **Mininet**, **OpenFlow**, and **Python** to monitor network traffic, collect flow statistics, and analyze network performance in real time.

## ✨ Features

* Real-time traffic monitoring
* Flow statistics collection
* Packet forwarding using OpenFlow
* Custom POX controller implementation
* Network topology simulation using Mininet
* Performance analysis of network traffic

## 🛠️ Technologies Used

* Python
* Mininet
* POX Controller
* OpenFlow 1.0
* Ubuntu/Linux

## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/your-repository.git
```

2. Navigate to the project directory.

```bash
cd your-repository
```

3. Install the required dependencies.

4. Start the POX Controller.

```bash
./pox.py controller
```

5. Run the Mininet topology.

```bash
sudo python topology.py
```

## ▶️ Usage

* Launch the POX controller.
* Start the Mininet topology.
* Generate network traffic between hosts.
* Monitor flow statistics and analyze traffic through the controller logs.
